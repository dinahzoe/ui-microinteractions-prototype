# Prototyp einer Browser-Extension zur Reduktion von wahrgenommenem Stress und Entscheidungsüberlastung

## Motivation
Durch berufliche Anforderungen und meist fest geregelte Arbeitszeiten geraten Beschäftigte häufig in einen Konflikt zwischen ausreichenden Pausen und einer effektiven Arbeitsweise. Daraus entstand die Idee, das Potenzial von UX-Design und Micro-Interaktionen zur kognitiven Entlastung zu nutzen.

## Zielsetzung
Ziel ist die Entwicklung eines funktionalen Prototyps, der zur Untersuchung der Auswirkungen auf Fokus und wahrgenommene Belastung dient und eine Verbindung zwischen UX-Design und Stressreduktion herstellt.

## Funktionaler Umfang
- Minimalistische Benutzeroberfläche  
- Stressreduzierende Micro-Interaktionen  
- Strukturierende und fokussierende UI-Elemente  

## Technischer Rahmen
- Browser-Extension (Chrome)  
- JavaScript und JSON  
- HTML und CSS für die Benutzeroberfläche
